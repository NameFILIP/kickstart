# CrowdCoin

## Getting Started

1. Create a following `credentials.js` file in the `ethereum` folder:

```
export const MNEMONIC_PHRASE =
  "xxx";
export const INFURA_ENDPOINT_RINKEBY =
  "https://rinkeby.infura.io/v3/yyy";
```

2. Run `npm install`
3. Run `npm run dev`
4. Go to http://localhost:3000
